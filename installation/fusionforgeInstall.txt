FusionForge Installation in Debian 7.6
======================================
1. Create the file “/etc/apt/preferences”
Put in the content from below:
Package: * Pin: release a=stable Pin-Priority: 700
Package: * Pin: release a=testing Pin-Priority: 650
Package: * Pin: release a=unstable Pin-Priority: 600

2. Edit the file “/etc/apt/sources.list”
Include the following line at the bottom:
deb http://ftp.de.debian.org/debian wheezy-backports main

3. Get and install FusionForge 5.3
apt-get update; apt-get install fusionforge-standard fusionforge-plugin-scmhook fusionforge-plugin-scmsvn fusionforge-plugin-scmgit fusionforge-plugin-scmcvs fusionforge-plugin-mediawiki fusionforge-plugin-moinmoin fusionforge-plugin-blocks ntp imagemagick php5-xsl

4. Restart Apache server
/etc/init.d/apache2 restart
